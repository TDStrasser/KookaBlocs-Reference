Part 1 - Working With KookaBlocs
==================================

In this Part 1 of the **KookaBlocs Reference Guide**, **KookaBlocs** in introduced, then instructions are given on how
it is installed on a personal computer, how **KookaBlocs** is used, 
and finally the conventions used by **KookaBlocs** are explained.
